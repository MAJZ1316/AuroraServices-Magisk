
# Aurora Services - Magisk

**A privileged extension for Aurora Apps**

Aurora Services is used to grant privileged permissions for [Aurora OSS Apps](https://gitlab.com/AuroraOSS) 
- Seamless app installs in backgound
- Panic trigger - WIP
